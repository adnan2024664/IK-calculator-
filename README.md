# IK Flooring Price Calculator

A web-based flooring price calculator built for internal use at IK Flooring.

## Purpose
The tool calculates flooring quotes based on:
- Room measurements
- Flooring type
- Installation costs
- Additional services

## Technologies Used
- HTML
- CSS
- JavaScript

## Description
This calculator helps employees quickly generate customer quotations by automatically calculating total costs based on flooring prices and fitting options.
